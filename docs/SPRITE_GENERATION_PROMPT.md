# 🎨 AI精灵图生成提示词模板

本文档提供了用于生成符合"校园大乱斗"游戏的精灵图的标准提示词模板。

---

## 📋 基础模板

```
生成一张游戏角色精灵表（Sprite Sheet）：

【必须要求】
1. 输出格式：4列 x 3行的网格（共12帧）
2. 总分辨率：768 x 1024 像素
3. 每帧大小：192 x 341 像素
4. 背景色：纯白色 RGB(255,255,255)
5. 角色风格：2D 像素画风格 / 街机风格

【帧结构说明】
- 第1行（Row 0）：待机/呼吸动画 - 4帧
  * 角色站立，轻微呼吸或摇晃
  * 用于游戏中的静止状态
  
- 第2行（Row 1）：跑动/移动动画 - 4帧
  * 角色向右跑动的连续动作
  * 帧序列：准备 → 跑步1 → 跑步2 → 跑步3
  
- 第3行（Row 2）：攻击/出拳动画 - 4帧
  * 角色出拳/攻击的连续动作
  * 帧序列：准备 → 出拳1 → 出拳2 → 收回

【角色描述】
{CHARACTER_DESCRIPTION}

【风格参考】
- 像素艺术（Pixel Art）风格
- 街机游戏风格（Arcade Style）
- 色彩鲜艳，对比度高
- 简洁有力的线条
- 动作夸张有趣
```

## 💡 提示词编写建议

### ✅ 做得好的做法
- **具体描述**：不要说"一个学生"，要说"穿着红色校服、戴着眼镜的数学天才"
- **颜色指定**：明确指定主色调和十六进制代码
- **动作细节**：描述每行的具体动作，而不是笼统地说"动画"
- **风格一致**：强调像素艺术和街机风格
- **分辨率明确**：总是指定 768x1024 和 192x341

### ❌ 避免的做法
- 模糊的描述（"看起来很酷"）
- 过于复杂的细节（会导致生成失败）
- 混合多种风格（坚持像素艺术）
- 忘记指定背景色（必须是纯白）
- 不清楚的帧结构

---

## 🔧 使用流程

1. **选择角色类型**
   - 使用学科角色示例，或
   - 使用自定义模板创建新角色

2. **填写角色描述**
   - 替换 `{CHARACTER_DESCRIPTION}` 或 `[角色描述]`
   - 确保包含所有必要信息

3. **提交给AI生成工具**
   - 复制完整的提示词
   - 粘贴到AI生成工具（BigModel、LMStudio等）
   - 等待生成

4. **验证生成结果**
   - 检查是否为 768x1024 分辨率
   - 验证 4x3 网格结构
   - 确认背景为纯白色
   - 测试在游戏中的显示效果

## 🎮 游戏中的应用

生成的精灵图将在游戏中用于：
- **待机行**：游戏中角色的静止状态
- **跑动行**：角色移动时的动画
- **攻击行**：角色攻击时的动画

确保动作流畅、帧数正确，以获得最佳游戏体验。
