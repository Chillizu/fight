# 校园大乱斗 - 项目开发计划

## 1. 项目概览

**项目名称**: 校园大乱斗 (Campus Fighting Game)
**类型**: 本地双人同屏 2.5D 横板网页格斗游戏
**技术栈**: HTML5 Canvas 2D + Vanilla JavaScript
**运行方式**: 纯前端本地化 (无需联网服务器)
**当前版本**: V0.7

### 核心特性

- 🎮 **本地双人对战**: 键盘分割屏幕控制
- 🤖 **AI角色生成**: 通过照片生成玩家专属格斗角色
- 📚 **文斗+武斗**: 13连击触发学科大招 → 时停答题释放技能
- ✨ **街机美学**: 粒子特效、屏幕震动、Hit Stop 顿帧

---

## 2. 功能规格

### 2.1 战斗系统

| 参数 | 值 | 说明 |
|------|-----|------|
| 游戏时长 | 60秒 | 倒计时 |
| 初始血量 | 1000 HP | 每位玩家 |
| 伤害 | 25 ±5 | 普通攻击 |
| 连击触发 | 13次 | 技能释放 |
| 答题时间 | 4秒 | 时停模式 |

### 2.2 控制键位

**P1 (左侧)**:
- A/D: 左右移动
- W: 跳跃
- J: 普通攻击
- K: 释放技能

**P2 (右侧)**:
- ←/→: 左右移动
- ↑: 跳跃
- 小键盘1: 普通攻击
- 小键盘2: 释放技能

### 2.3 13种学科技能

| 学科 | 技能名 | 效果 |
|------|--------|------|
| 生物 | 变大 | 角色放大1.5倍，伤害+30% |
| 化学 | 毒气 | 紫绿色毒雾，持续掉血 |
| 物理 | 禁锢 | 冻结对手4秒 |
| 语文 | 反转 | 反向控制对手 |
| 数学 | 狂暴 | 攻击速度×2 |
| 英语 | 无敌 | 4秒无敌 |
| 政治 | 沉默 | 禁止对手技能 |
| 历史 | 回血 | 回复300血 |
| 地理 | 陨石 | 全屏群伤500 |
| 体育 | 加速 | 移动速度×1.5 |
| 信息 | 黑客 | 偷取对手技能 |
| 音乐 | 眩晕 |  opponent can't move |
| 美术 | 幻象 | 制造假身 |

## 3. 技术架构

### 3.1 文件结构

```
Fight_/
├── index.html           # 主游戏文件
├── js/
│   ├── config.js       # 配置常量 ⭐
│   ├── game.js        # 游戏循环、碰撞检测
│   ├── player.js      # 玩家类、动画
│   ├── ui.js         # UI渲染
│   ├── ui_extend.js  # AI集成、背景移除
│   ├── subjects.js   # 13技能定义
│   └── particles.js # 粒子特效
├── assets/
│   ├── backgrounds/  # 游戏背景
│   └── sprites/      # 角色精灵表
└── tools/
    ├── verify_system.py     # 系统检查
    └── sprite_processor.py # 精灵表处理
```

### 3.2 精灵表规格

- 格式: 4列 × 3行 (共12帧)
- 分辨率: 768×1024 像素 (每帧 192×341)
- 背景: 纯白色 RGB(255,255,255)

**帧结构**:
- 第1行: 待机 (Idle) - 4帧
- 第2行: 跑动 (Run) - 4帧
- 第3行: 攻击 (Attack) - 4帧
