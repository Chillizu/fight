本地开发服务器启动指南
💡 提示：游戏需要在本地服务器上运行才能正常加载所有资源。请选择下方任一方法启动服务器。
🐍 方法1: Python (推荐)
步骤:
1. 打开命令行/终端
2. 进入游戏目录
3. 运行: python -m http.server 8000
4. 访问: http://localhost:8000

ℹ️ 需要已安装Python。如未安装，
请访问: https://www.python.org

⚡ 方法2: 启动脚本 (最简单)
步骤:
1. 在游戏目录找到 start_server.bat
2. 双击运行即可
3. 等待浏览器自动打开
4. 开始游戏!
📦 方法3: Node.js
步骤:
1. 打开命令行
2. 进入游戏目录
3. 运行: node start_server.js
4. 访问: http://localhost:8000

ℹ️ 需要已安装Node.js。如未安装，
请访问: https://nodejs.org

⚠️ 重要: 服务器启动后，请在浏览器访问 http://localhost:8000 来玩游戏。
